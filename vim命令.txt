（一）进入
输入：vim filename
vim的特征：vim是一个多模式编辑器，默认处在命令模式
vim有三种常见的模式：命令模式、插入（编辑）模式、底行模式

（二）退出（底行模式操作）
命令模式-》底行模式：“shitf+：”
:w    保存
:wq  保存退出
:q     直接退出
:q!    不保存，强制退出
命令模式下大写ZZ也是保存退出


（三）输入模式（在命令模式下操作）
i    从光标所在位置前面开始插入    
I    在当前行首插入    
a   从光标所在位置后面开始输入    
A   在当前行尾插入    
o   在光标所在行下方新增一行并进入输入模式    
O  在当前上面一行插入


（四）移动（定位）光标（在命令模式下操作）
gg    到文件第一行    
G      到文件最后一行   (Shift + g)    
nG  （n shitf+g）将光标定位到任意n行
0       将光标定位到行首(数字0)
^      将光标定位到行首非空格的位置（锚点）    
$       行尾
w      按照单词进行后移
b      按照单词进行前移
vim  文件名  +n     可以将光标直接定位在第n行，在gcc编译出错后可以快速                          定位
!vim  可以执行上一次的vim操作，比如 vim file.c  +9


（五）复制和粘贴（在命令模式下操作）
yy    复制整行内容    
nyy   可以直接复制多行，如3yy就是复制三行的内容
yw   复制当前光标到单词尾内容    
p      粘贴，光标在那里就粘贴到下一行
np：n代表个数，可以重复的粘n行


（六）删除（在命令模式下操作）
dd   删除光标所在行,如果再按p的话粘贴的是删除的行，功能更像剪切
ndd 批量删除一批相连续的行
dw  删除一个单词    
x     删除光标所在字符    nx  删除光标所在字符向后n个字符
n shift+x    删除光标所在字符向前n个字符
u    撤销上一次操作    
ctrl + r    撤销取消操作（反撤销）
shift+~   可将光标处的字母转换大小写
h，j，k，l分别代表光标的左，下，上，右移
s     替换    


（七）查找（在底行模式下操作）
/： 查找文本，n选择下一个
?： 查找文本，n选择上一个  
n    向下查找    
N   向上查找
set nu  调出行号
set nonu  取消行号
vs  文件名：可以打开另一个文件，此时光标在哪里，就编辑哪个文件
（下面两条为命令模式下操作）
ctrl + w  再 +   h或者l     可以切换编辑的文件
ctrl + 双击两次w     也可以切换编辑的文件

（八）块操作
v    块选择    
ctrl + v   列块选择


（九）运行py文件
在vim内，在命令模式下按F5
退出vim，在终端输入 pyhton3 filename


vim配置：
一般.vimrc每个账号下都有一个，也就是说个人配置个人，彼此之间互不影响

翻译的四个阶段：预处理/编译/汇编/链接
预处理：头文件展开，去注释，宏替换，条件编译
编译：将C代码翻译形成汇编代码
汇编：将汇编形成目标二进制文件
链接：将目标文件和系统库进行链接形成可执行程序
